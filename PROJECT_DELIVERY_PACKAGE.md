## 🚀 Quick Start Guide

### **Step 1: Build the Project**
```bash
npm install
npm run build
```

### **Step 2: Deploy to IONOS**
1. Upload everything from the `dist` folder to your web root
2. Add the `.htaccess` file (instructions in build-instructions.md)
3. Test your site!
